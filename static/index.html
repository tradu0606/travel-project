US National Parks API
Method url Parameter Type What it does
https://tanyasproject.azurewebsites.net/
GET limit - Query Number of documents to return per request. Default limit equals 5.
page - Query Next "limit" result will start from number "page". Default page equals 0 		Query works for all get requests.
GET /parks/ limit, page - Query, Integer Display a collection of all US National Parks
GET /parks/name/:name :name - String Search parks by name, shows alerts and newspaper for this park
GET /parks/state/:state :state - String Search parks by state, shows alerts and newspapers for this park
GET /news/ limit, page - Query, Integer Display all news for all parks, shows 5 per request
GET /news/title/:title :title - String The title of the alert Search alerts by title
POST /news/ body - {Body} Should contain an object to add to the database Add new newspaper to the database. Requires a key to authenticate
PUT /news/update/:name/ body - {Body} Should contain an object to add to the database
:name - String Name of a field to update		 Add new newspaper to the database. Requires a key to authenticate
DELETE /news/delete/:title :title - String Name of the field to update Remove news from the database
GET /alerts/ limit, page - Query, Integer Display all alerts, shows 5 per request
GET /alerts/category/:type :type - String The name of the category Display all alerts of a certain category, shows 5 per request
GET /alerts/title/:title :title - String The title of the alert Search alerts by title
POST /alerts/add/ body - {Body} Should contain an object to add to the database Add new newspaper to the database. Requires a key to authenticate
PUT /alerts/uptate/:title body - {Body} Should contain an object to add to the database
:title - String Name of the field to update		 Update a field/s in the database. Requires a key to authenticate
DELETE /alerts/delete/:title :title - String Name of the field to update Remove alert from the database
