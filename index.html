US National Parks API
Method url Parameter Type What it does
GET localhost:8080/parks/ query
limit - number of documents to return per request. Default limit equals 5.
page - next "limit" result will start from number "page". Default page equals 0 		Display a collection of all US National Parks
GET localhost:8080/parks/name/:name :name - String Search parks by name, shows alerts and newspaper for this park
GET localhost:8080/parks/state/:state :state - String Search parks by state name, shows alerts and newspaper for this park
GET localhost:8080/news/ query:
limit - number of results to return per request. Default limit equals 5
page - next "limit" result will start from number "page". Default page equals 0		 Display all news for all parks, shows 5 per request
POST localhost:8080/news/ body - should contain an object to add to database Add new newspaper to database. Require a key authentication
PUT localhost:8080/news/update/:name/ body - should contain an object to update in database
:name - name of a field to update		 Update info in a database. Require a key authentication
GET localhost:8080/alerts/ query:
limit - number of results to return per request. Default limit equals 5
page - next "limit" result will start from number "page". Default page equals 0		 Display all alerts for all parks, shows 5 per request
GET localhost:8080/alerts/category/:type :type - String, the name of category Display all alerts certain category, shows 5 per request
POST localhost:8080/alerts/ body - should contain an object to add to database Add new alert in database. Require a key authentication
PUT localhost:8080/alerts/ body - should contain an object to update in database
:title - name of a field to update		 Update an alert in a database. Require a key authentication
