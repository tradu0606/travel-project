US National Parks API
Method url Parameter Type What it does
GET limit - Query Number of documents to return per request. Default limit equals 5.
page - Query Next "limit" result will start from number "page". Default page equals 0 		Query works for all get requests.
GET localhost:8080/parks/ limit, page - Query, Integer Display a collection of all US National Parks
GET localhost:8080/parks/name/:name :name - String Search parks by name, shows alerts and newspaper for this park
GET localhost:8080/parks/state/:state :state - String Search parks by state, shows alerts and newspapers for this park
GET localhost:8080/news/ limit, page - Query, Integer Display all news for all parks, shows 5 per request
GET localhost:8080/news/title/:title :title - String The title of the alert Search alerts by title
POST localhost:8080/news/ body - {Object} Should contain an object to add to the database Add new newspaper to the database. Requires a key to authenticate
PUT localhost:8080/news/update/:name/ body - {Object} Should contain an object to add to the database
:name - String Name of a field to update		 Add new newspaper to the database. Requires a key to authenticate
DELETE localhost:8080/news/delete/:title :title - String Name of the field to update Remove news from the database
GET localhost:8080/alerts/ limit, page - Query, Integer Display all alerts, shows 5 per request
GET localhost:8080/alerts/category/:type :type - String The name of the category Display all alerts of a certain category, shows 5 per request
GET localhost:8080/alerts/title/:title :title - String The title of the alert Search alerts by title
POST localhost:8080/alerts/add/ body - {Object} Should contain an object to add to the database Add new newspaper to the database. Requires a key to authenticate
PUT localhost:8080/alerts/uptate/:title body - {Object} Should contain an object to add to the database
:title - String Name of the field to update		 Utate a field/s in the database. Requires a key to authenticate
DELETE localhost:8080/alerts/delete/:title :title - String Name of the field to update Remove alert from the database
